# 問題1: 数列の合計と偶数 - ヒント

## 問題概要
配列の全要素の合計と偶数の個数を求め、指定された形式で結果を返す。

## ヒント段階

### 基本ヒント
1. 配列の要素の合計を計算する方法を考えましょう
2. 配列の中から偶数の要素だけを見つける方法を考えましょう
3. 最終的に「合計：XX、偶数の個数：YY」という形式の文字列を作る必要があります

### 詳細ヒント（必要な場合）

**合計の計算方法**:
- 単純なループを使う方法
```typescript
let sum = 0;
for (const num of numbers) {
  sum += num;
}
```
- 配列メソッドを使う方法 (`reduce`)
```typescript
// reduceの基本構文
// reduce(コールバック関数, 初期値)
// コールバック関数は (積み上げ値, 現在の要素) => 新しい積み上げ値
```

**偶数の個数を数える方法**:
- 偶数かどうかは `num % 2 === 0` で判定できます（2で割った余りが0）
- ループを使って偶数をカウントする方法
```typescript
let count = 0;
for (const num of numbers) {
  if (num % 2 === 0) {
    count++;
  }
}
```
- 配列メソッドを使う方法 (`filter`)
```typescript
// filterの基本構文
// filter(要素 => 条件式)
// 条件式がtrueの要素だけの新しい配列を返す
```

**文字列の作成方法**:
- テンプレートリテラルを使う方法
```typescript
// バッククォート(`)を使い、${変数名}で変数を埋め込める
`合計：${sum}、偶数の個数：${count}`
```

## 学習ポイント
- 配列の`reduce()`メソッド - 配列を単一の値に集約
- 配列の`filter()`メソッド - 条件に合う要素だけを抽出
- テンプレートリテラル - 文字列内に変数を埋め込む
- アロー関数 - 簡潔に関数を定義する
